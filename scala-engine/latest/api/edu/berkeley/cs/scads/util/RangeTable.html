RangeTable

class RangeTable [KeyType, ValueType] extends AnyRef

Range Table data structure. Allows to split the range -inf to inf in arbitrary sub-ranges and assign values per sub-range. The operations are thread-safe

Constructor assigns values to the range -inf to inf. This range can than be split by split(key). Keys are compared with the given comparator

TODO RangeTable should be fully immutable (also for adding values). Makes programming the protocols easier
