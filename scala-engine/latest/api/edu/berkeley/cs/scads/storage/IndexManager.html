IndexManager

trait IndexManager [BulkType <: AvroPair] extends Namespace with Protocol with RangeKeyValueStoreLike[IndexedRecord, IndexedRecord, BulkType] with Serializer[IndexedRecord, IndexedRecord, BulkType] with ZooKeeperGlobalMetadata with RecordStore[BulkType]

An IndexManager is intended to provide index maintainence for AvroPair namespaces. the methods below are not intended to be thread-safe

    Create a new index over this pair type, with the given name.

    Create a new index over this pair type, with the given name. The fields given MUST be (1) unique and (2) part of the key schema or the value schema. The ordering of fields determines the layout of the index schema.
